Veranstalter Gutscheine

Um unseren Urlaubern zu ermöglichen, bei einer Buchung direkt einen von Ihnen ausgegebenen Gutschein zu verwenden, können Sie hier die dafür benötigten Daten zur Verfügung stellen. Laden Sie dazu hier eine .csv Datei mit den folgenden Spalten hoch:

  • email: Die bei der Buchung verwendete E-Mail Adresse des Urlaubers
  • amount: Der Wert des Gutscheins
  • currencyCode: Das 3-Buchstaben Kürzel der Währung, in der der Gutschein ausgestellt wurde
  • code (optional): Der Gutschein-Code, den der Urlauber angeben muss. Dieses Feld kann auch leer gelassen werden, dann kann der Gutschein durch den Urlauber ohne code eingelöst werden.
  • validUntilDate (optional): Das Datum, bis zu dem der Gutschein seine Gültigkeit behält. Wenn diese Spalte leer gelassen wird, läuft der Gutschein bis zu seiner Einlösung nie ab. Das Datum muss dabei in folgendem Format angegeben werden: YYYY-MM-DD
  • tourOperatorInternalNumber (optional): Ihre Vorgangsnummer der stornierten Buchung, für die der Gutschein ausgestellt wurde

Optionale Spalten dürfen dabei nicht ganz weggelassen werden, sondern müssen leer sein. Die Reihenfolge der Spalten muss eingehalten werden.

Die Gutscheine können sofort von den Urlaubern verwendet werden.

Beispiel:

email,amount,currencyCode,code,validUntilDate,tourOperatorInternalNumber
urlauber1@test.com,233.12,EUR,COR-12345,2021-12-31,123456789
urlauber2@test.com,11.00,CHF,,,
Keine Datei ausgewählt
